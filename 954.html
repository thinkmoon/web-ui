windows环境git（github) 使用代理ssh/http

修改或增加~/.ssh/config

ProxyCommand connect -S 127.0.0.1:10808 %h %p

其中-S代表使用socks代理。如果是http模式，则为-H
后接代理地址，如127.0.0.1:10808

如果你只想修改特定的情况做配置，可以增加如下内容

Host github.com
 User git
 Port 22
 Hostname github.com
